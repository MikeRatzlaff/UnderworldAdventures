Development

Welcome to the Underworld Adventures development section!

The Underworld Adventures source code is hosted on GitHub. More infos about the source code can be found there. If you want to submit a bug report, you can use the Github project's issue tracker.

A Developer's log is available that has log entries of what happened with the project so far.

In case you want to contact me, just use the email address shown in the bottom of this page.

Plans

The plan for the project was to bring Ultima Underworld 1 to modern operating systems, by implementing the engine and the game logic. After that the project would try to support the more complex Ultima Underworld 2. Finally the idea was to create a game development studio to create new games in the style of the Ultima Underworlds.

Nowadays you can play uw1 and uw2 easily using Dosbox, and you still can get it from gog.com. This project still is up for these two goals:

  • to keep and provide the history of game development for uwadv
  • to showcase what the project has reached at this point
I occasinally will work on the Underworld Adventures code, in order to improve my C++ and graphics programming skills and to dig into and hack the original game files. You're welcome to join!

Team

Currently there is no team, only me:

  • Michael Fink (vividos), located in Germany; project founder and developer.

Previous team members are:

  • Willem Jan Palenstijn (wjp), lives in the Netherlands; his main task was Linux porting.
  • Dirk Manders (phlask), located in the Netherlands and contributed the cutscene code and the intro cutscene in Lua, and the character creation screen.
  • Martin Shetty (QQtis), from Washington, DC, worked on digitally rearranging the Ultima Underworld MIDI soundtrack using a software synth.
  • Kasper Fauerby (Telemachos), located in Denmark, helped out with the collision detection code and ideas.
  • Aisha Fenton, lives in New Zealand and was working on the MacOS X port.
  • Jim Cameron (hairyjim) from the TSSHP project, located in the UK, whose formats file initiated this project.
  • All others who contributed or influenced the project in a way. Thank you!

#uwadv IRC channel

The project has a (once lively) official IRC channel. Just connect to irc.freenode.net and join #uwadv. wjp's chatbot 'exultbot' still collects and hosts the archive, here: #uwadv logs. Exultbot also understands some commands. Just say "?help" in the channel to get an overview of the commands.
